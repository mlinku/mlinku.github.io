HTML 标题（Heading）是通过标签来定义的。

 <h1>这是一个标题</h1>
 <h2>这是一个标题</h2>
 <h3>这是一个标题</h3>

因为用户可以通过标题来快速浏览您的网页，所以用标题来呈现文档结构是很重要的。
应该将 h1 用作主标题（最重要的），其后是 h2（次重要的），再其次是 h3，以此类推

HTML 图像是通过标签 来定义的.

alt 属性用来为图像定义一串预备的可替换的文本。
替换文本属性的值是用户定义的。
在HTML 4中 align 属性已废弃，HTML5 已不支持该属性，可以使用 CSS 代替。
